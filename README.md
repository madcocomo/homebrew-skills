# homebrew-skills

这个仓库用于保存我自制的 pi assets 源文件，主要包括：

- skills
- extensions
- 相关提示词/文档/实验材料

## 仓库定位

这里保存的是**源文件**与历史变更记录。

某些内容可能会被复制到全局 pi 目录中生效，例如：

- `~/.pi/agent/extensions/...`

但这些全局位置应视为**运行中的副本**，不应默认当作唯一真实来源。

## 工作约定

- 优先在本仓库修改源文件
- 需要生效时，再同步到全局 pi 目录
- 提交记录用于保留这些自制 skill / extension 的演进历史

## 说明

创建这个 README 的目的，是避免未来把本仓库误认为临时目录、生成产物目录，或在不了解用途的情况下误删、误改、误覆盖。
