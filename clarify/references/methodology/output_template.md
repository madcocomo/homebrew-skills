# 输出模板

## 模式文档结构

每个模式应包含以下部分：

```
# [模式名称]

## 目的
[一句话说明这个模式解决什么问题]

## 问题根源
[为什么人类在处理这类信息时容易产生理解困难或错误]

## 描述
[模式的详细说明]

## 适用场景
- [场景1]
- [场景2]
- ...

## 实施方法
1. [步骤1]
2. [步骤2]
3. [步骤3]

## 示例

### 正面示例
[展示正确应用模式的例子]

### 反面示例
[展示未应用模式导致问题的例子]

## 效果
- **理解效率提升**：具体说明如何降低认知负担
- **错误发现机制**：具体说明如何帮助发现错误

## 组合建议
[与其他模式组合使用的建议]
```

## 三、应用指南
- 如何选择合适的模式
- 模式组合示例
- 常见场景的推荐模式

## 四、模式索引
