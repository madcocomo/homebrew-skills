# 数据源指南

## 数据源类型

### 1. 学术论文

- **来源**：arXiv, ACL, NeurIPS, ICML, ICLR等AI会议
- **类型**：研究论文、综述文章
- **优先级**：高
- **可信度**：高
- **示例**：
  - 可解释AI (XAI) 相关论文
  - 人机交互 (HCI) 关于信息呈现的研究
  - 认知科学关于理解与推理的论文

### 2. 技术博客与行业报告

- **来源**：知名AI研究机构博客、行业分析报告
- **类型**：技术解读、行业分析
- **优先级**：高
- **可信度**：中高
- **示例**：
  - OpenAI、Anthropic、Google AI博客
  - Gartner、McKinsey技术报告

### 3. 官方文档与指南

- **来源**：AI模型官方文档、API文档
- **类型**：使用指南、最佳实践
- **优先级**：高
- **可信度**：高
- **示例**：
  - Anthropic Claude文档
  - OpenAI GPT指南

### 4. 开源项目与社区讨论

- **来源**：GitHub, Stack Overflow, Reddit
- **类型**：项目文档、社区讨论
- **优先级**：中
- **可信度**：中
- **示例**：
  - Prompt工程开源仓库
  - AI对齐研究社区讨论

### 5. 维基与百科

- **来源**：Wikipedia, 各类百科
- **类型**：概念定义、背景知识
- **优先级**：中
- **可信度**：中
- **注意**：需交叉验证

## 需谨慎使用的信息源

- **个人博客（非专业人士）**：缺乏同行评审
- **未经验证的社交媒体观点**：可信度低
- **明显带有商业推广目的的内容**：可能有偏见

## 信息记录格式

每个收集到的信息源应记录：

```markdown
## [SRC-XXX] 信息源标题

**来源URL**: [URL]
**来源类型**: [学术论文/技术博客/官方文档/...]
**收集时间**: [YYYY-MM-DD]
**发布机构/作者**: [名称]

**核心内容摘要**:
[简要总结关键内容]

**相关模式点**:
- 关联的模式类别：[可理解性/错误发现]
- 潜在模式名称：[可能的模式]

**可信度评估**: [高/中/低]
**评估理由**:
[说明评估依据]
```

## 质量检查清单

- [ ] 每个信息源都有明确的URL或引用
- [ ] 内容与AI呈现清晰度主题相关
- [ ] 包含足够的细节支持模式识别
- [ ] 评估了来源的可信度
- [ ] 标注了可能的模式关联
