# 输出模板

## 模式描述模板

```markdown
# [模式名称]

**分类**: [功能分类] - [层次分类]

**目的**: [简明描述模式要解决的问题]

**描述**:
[模式的详细描述，包括原理和核心要素]

**适用场景**:
- 场景1: [具体说明]
- 场景2: [具体说明]
...

**实施指南**:
1. 步骤1: [具体说明]
2. 步骤2: [具体说明]
3. 步骤3: [具体说明]
...

**示例**:

*正面示例*:
```
[展示正确应用模式的示例]
```

*反面示例*:
```
[展示错误应用模式的示例]
```

**注意事项**:
- 注意点1
- 注意点2
...

**与其他模式的关系**:
- 依赖: [相关模式]
- 增强: [相关模式]
- 冲突: [相关模式]
```

## 格式规范

### 标题层级
- #: 文档标题
- ##: 主要章节
- ###: 子章节
- ####: 小节

### 列表格式
- 无序列表: 使用 "-"
- 有序列表: 使用 "1."
- 任务列表: 使用 "- [ ]"

### 代码块
- 行内代码: 使用 "`"
- 代码块: 使用 "```"

### 引用
- 块引用: 使用 ">"
- 链接: 使用 "[文字](URL)"
