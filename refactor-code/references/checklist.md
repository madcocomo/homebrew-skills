# 代码重构检查清单

## 重构前
- [ ] 理解现有实现和依赖关系
- [ ] 确认现有测试覆盖范围
- [ ] 备份或记录当前行为

## 重构中
- [ ] 小步修改，频繁测试
- [ ] 保持 API 行为不变
- [ ] 遵循项目代码风格
- [ ] 无新增硬编码值

## 重构后
- [ ] 所有测试通过
- [ ] 代码更简洁/可维护
- [ ] 无重复代码
- [ ] 命名更清晰

## 验证命令
```bash
# 运行测试
cd avia-base && mvn test

# 运行特定测试
mvn test -Dcucumber.filter.name=".*场景名.*"
```
